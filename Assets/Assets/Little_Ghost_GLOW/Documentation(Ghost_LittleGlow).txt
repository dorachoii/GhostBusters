These cute little ghosts are born to scare you!!! (Just joking,..its just a cute funny little ghosts.Take them)

This pack contains 3d Animated Little Ghosts with many Color variations.Pack comes with many texture variations and a custom Shader,which you can use to make any thing looks like ghost thing. You can use this Asset commercially and non-commercially.This model is optmized for using in pc, mobile and AR games.

Note: Ghost and Materials are highly customizable.You can create Unlimted variations of Ghost's by changing material properties and Meshes.For quick use, 20 ghosts are included in prefabs.Also, this asset uses Shadergraph with Built-in support.

Technical Details:-
		- Model(s) : Little Ghost, Hat, other props (Eyes, Mouth.etc), Props x25
		- Model format : (.FBX)
		- Polygon count : 7830 (Including Ghost, Hat and other props), Props (12-2000 polys)
		- Unwrapped : Yes (Ghost only)
		- Animations : 12 (Idle x 2, Fly/Run x 2, Hide/Death, Wake, Attack x 2, Dance x 3, Look)
		- Materials : 13+ (CUSTOMIZABLE), 10+ for Props
		- Used Material(s) : Custom(Made with Shadergraph 12.1.11)
		- Prefabs : 20+ (Ghost), 100+(Props)
